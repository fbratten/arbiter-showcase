## What is arbiter?

arbiter validates MCP servers against the MCP specification. It checks protocol compliance, tool quality, error handling, security, and LLM usability. Returns maturity scores with actionable fix packs.

- 5 MCP tools
- 4 validation profiles (lenient/standard/strict/paranoid)
- 5 badge levels (CERTIFIED to NON_COMPLIANT)
- 7 fix packs with effort estimates
- 177 passing tests
